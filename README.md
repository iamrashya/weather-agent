# Weather Agent 🌍

A Python agentic app that fetches and displays real-time weather for the top 10 most populous countries using the free Open-Meteo API. No API key required.

## How to Run

Step 1: `python -m venv venv`

Step 2: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux)

Step 3: `pip install -r requirements.txt`

Step 4: `python agent.py`

## Data Source

[Open-Meteo](https://open-meteo.com) — free, no API key required.

## Output

Displays a live weather table in the terminal with temperature, wind speed and conditions for each country.
